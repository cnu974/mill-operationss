# Mill Operations

Rice mill operations management application - Paddy intake, production, sales, and finance.

## Features
- Dashboard with live inventory & financial data
- User authentication via Supabase  
- Procurement, Production, Sales, and Finance modules
- Radix UI components

## Deployment
Live at: https://mill-operationss.vercel.app

## Tech Stack
- React 19
- TanStack Router
- TanStack React Query
- Supabase
- Tailwind CSS
- Vite
